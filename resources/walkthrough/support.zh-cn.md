# 支持项目

DeepSeek Pilot 是 MIT 开源且免费的。如果它为你节省了时间或金钱，以下是一些回馈方式：

- ★ 在 [GitHub](https://github.com/setsey/deepseek-pilot) 上 Star 仓库
- 💖 [在 GitHub 上赞助](https://github.com/sponsors/setsey) — 即使是小额定期赞助也能帮助分担测试费用
- 🐞 [提交问题或功能请求](https://github.com/setsey/deepseek-pilot/issues)
- 🗣 向任何在 DeepSeek 上运行长会话的人分享此扩展

特别欢迎附带复现步骤和 **DeepSeek Pilot：显示日志** 输出的 Bug 报告。
