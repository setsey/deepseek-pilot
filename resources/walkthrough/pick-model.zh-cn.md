# 选择 DeepSeek Pilot 模型

打开 Copilot Chat（`Ctrl+Shift+I`）并从模型选择器中选择一个模型：

- **V4 Pro（思考模式）**：复杂代码审查、故障排查
- **V4 Pro**：功能强大、低延迟、无推理链
- **V4 Flash（思考模式）**：经济实惠的深度思考分析
- **V4 Flash**：最便宜、最快、简单任务

如果提供者已存在但尚未完全配置，请从选择器菜单或命令面板中打开 **DeepSeek Pilot：管理提供者**。

思考模式变体还提供了一个按模型切换的**推理深度**选择器，支持 `high` 和 `max`。

**视觉支持：** 将图像拖入聊天中。扩展会自动通过支持视觉的模型代理处理图像，然后将文本描述发送给 DeepSeek。

**设为效用模型：** 从**管理提供者**中选择**设为 Copilot 主效用模型**（或*轻量*变体），将 Copilot 的后台流程——标题、摘要、提交信息、意图识别——路由到 DeepSeek Flash。便宜、快速、无需思考。

**状态栏：** 在一个组件中显示上下文窗口饱和度、会话 token 消耗、估算费用和平台余额。悬停可查看 KV 缓存感知的压缩建议。
